Revisão de HTML5

1. Definições de tag

Algumas tags de HTML5:

  • p/p serve para adicionar parágrafos.
  • h1/h1 serve para adicionar títulos.(Obs: Essa tag é enumerada apenas o número 6.)
  • b/b serve para deixar seu texto em negrito.
  • em/em é usada para deixar o texto em ênfase.
  • li/li é usada para para fazer listas junto com a tag ol/ol e ul/ul
  • sup/sup para deixar as frase em cima das frases e sob/sob para deixar em baixo.
  • big/big para deixar o texto grande e small/smll para deixá-la pequena.
  • ins/ins para deixar o texto inerido.
  • mark/mark para marcar o texto. Porém para mudar a cor texto será necessário usa o CSS3.
  • del/del para deixar o texto excluído ou deletado.
  • dt/dt é uma tag de descrição acompanhada com dd/dd
  • address/address serve para colocar endereços.

2. Colocando ícones e imagens em um site.

Para colocar imagens é necessário usar a tag img src="" alt=""
Por exemplo: img src="logohtml5.png" alt="logo HTML5" img src="imagens/logocss3.png" alt="logo CSS3" img src="https://logospng.org/download/javascript/logo-javascript-512.png" alt="Logo JavaScript"

Para colocar ícones na imagens basta colocar a tag link rel="shortcut icon" href="" type="image/x-icon"

3. links.

é usada a seguinte tag para colocar links.

a href="" rel="prev">
